# CI/CD Final Project

## Project Name: CI/CD Pipeline with GitHub Actions, Tekton, and OpenShift

## Overview
This project demonstrates a complete CI/CD pipeline using:
- **GitHub Actions** for automated linting and unit testing
- **Tekton Pipelines** for task management
- **OpenShift** for container deployment

## Application
A Python Flask-based RESTful counter service.

## Tech Stack
| Tool | Purpose |
|------|---------|
| Python / Flask | Backend web application |
| Flake8 | Python linting |
| Nose | Unit testing framework |
| GitHub Actions | CI workflow automation |
| Tekton | CD pipeline tasks |
| OpenShift | Cloud deployment platform |

## How to Run Tests
nosetests -v --with-spec --spec-color --with-coverage --cover-package=app
